# Developer Instructions #

## Build Environment Install ##

`npm install`

## Build Commands ##

Build everything:

```
npm run build
```

Build CSS only:

```
npm run build:css
```

Convert readme.txt to README.md:

```
npm run build:readme
```

Build plugin.zip:

```
npm run build:zip
```

## Watch Commands ##

Watch CSS:

```
npm run watch
```
